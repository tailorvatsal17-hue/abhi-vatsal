## How It Works

### The Image Rendering Flow

```
Database Entry
  (e.g., "/images/services/plumbing.jpg")
         ↓
Pug Template Calls Helper
  (e.g., getImageUrl(service.image, 'service'))
         ↓
Helper Validates Path
  ✓ Is it null/empty? → Return fallback placeholder
  ✓ Starts with /? → Return as-is (local file)
  ✓ Starts with http? → Return as-is (external URL)
  ✓ Other? → Prepend /
         ↓
HTML <img> Tag Gets URL
  (e.g., src="/images/services/plumbing.jpg")
         ↓
Browser Requests Image
  ✓ File exists? → Shows image ✓
  ✓ File missing? → Shows fallback ✓
  ✓ External URL? → Shows remote image ✓
```

### Example Usage in Pug

```pug
// 1. Get image URL with helper
- var imageUrl = getImageUrl(service.image, 'service')

// 2. Render in HTML
img(src=imageUrl, alt=service.name, style="width: 100%; height: 150px; object-fit: cover;")

// Result:
// If DB has: /images/services/plumbing.jpg
//   → <img src="/images/services/plumbing.jpg" ...>
// 
// If DB has: null
//   → <img src="https://via.placeholder.com/400x300?text=Service" ...>
//
// If DB has: https://cdn.example.com/image.jpg
//   → <img src="https://cdn.example.com/image.jpg" ...>
```

## Files Modified

### app/app.js
**Lines 12-42:** Added image helper functions to app.locals

```javascript
// Image helper functions for Pug templates
const getImageUrl = (imageUrl, type = 'service') => {
  const fallbacks = {
    service: 'https://via.placeholder.com/400x300?text=Service',
    partner: 'https://via.placeholder.com/400x300?text=Professional',
    customer: 'https://via.placeholder.com/150x150?text=Profile',
    category: 'https://via.placeholder.com/200x200?text=Category',
    work: 'https://via.placeholder.com/400x300?text=Portfolio'
  };

  if (!imageUrl || imageUrl.trim() === '') {
    return fallbacks[type] || fallbacks.service;
  }
  if (imageUrl.startsWith('/') || imageUrl.startsWith('http')) {
    return imageUrl;
  }
  return '/' + imageUrl;
};

const parseWorkImages = (workImages) => {
  if (!workImages) return [];
  return workImages
    .split(',')
    .map(img => img.trim())
    .filter(img => img.length > 0)
    .map(img => getImageUrl(img, 'work'));
};

// Make helpers available to all Pug templates
app.locals.getImageUrl = getImageUrl;
app.locals.parseWorkImages = parseWorkImages;
```
